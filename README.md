# tiktok-remote
### Install library
```
  pip install require.txt
```
### Go to pytorch folder
```
  cd pytorch
```

### Create data
#### Update label in file create_data.py
```
   python create_data.py
```
### Train data
#### Update path to save model, path to data
```
   python train.py
```

### Test
``` 
   python test.py
```

### Apply to tiktok remote
```
  python app.py
```
